react 这个东西说起来就像玩积木，需要什么拿什么，拼起来用，所以它需要结合Redux React-router这些库来完成项目的搭建。

首先要下载react及react-dom。

由于要打包所以webpack 也是不能少的。

用es6语法，所以要设置babel

我是用less写的要下载less的loader

要用到browser的热更新，所以也要下载，配合webpackDevMiddleware和
webpackHotMiddleware使用

webpack里就简单的配置一下就可以了。

值得一提的是根目录下面得配置一个.babelrc的文件，不然webpack.config文件就不能用es6语法了

browser也随便配置一下

然后要配置package,重点看scripts里,npm-run-all可以组合你的命令哦！

好了，一切准备就绪，开始码代码
在根目录下创建src文件夹后创建index.js里面引入我们的react

在根目录下创建index.html
引入打包好的bundle.js

然后命令行输入,项目就运行起来了，hello world!
